Mario Game in C

A classic 2D platformer game inspired by Super Mario, developed from scratch using the C programming language.
🎮 Game Features

    Core Gameplay: Smooth character movement using keyboard inputs, mimicking the classic platformer experience.

    Level Design: Features two distinct game maps to explore.

    Game Mechanics: * Interactive environment: Collect power-ups and avoid hazards.

        Scoring System: Real-time points tracking based on in-game actions.

    User System: Includes a registration/login system to track users.

    Audio Integration: Immersive gameplay with background music and sound effects for game events (win, game over, collecting items).

🛠 Technical Details

    Language: C

    Platform: Console-based rendering

    Architecture: Modular code structure using myfuncs.h for efficient function management.

🚀 How to Run

    Clone this repository:
    Bash

    git clone https://github.com/arnili64/mario-projectBp.git

    Compile the source code using any C compiler (e.g., GCC):
    Bash

    gcc main.c myfuncs.c -o mario_game

    Run the executable:
    Bash

    ./mario_game
